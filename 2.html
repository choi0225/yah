CSS

CSS이란

CSS는 Cascading Style Sheets의 약자로.
HTML,XHTML,XML 같은 문서에 사용하는 스타일을 꾸밀 때 사용하는 스타일 시트 언어 입니다.
HTML로 문서의 뼈대를 만들면
CSS는 이 문서의 화장을 맡고 있습니다.
글꼴이나,배경색,너비와 높이,위치등을 지정하거나
웹 브라우저,스크린 크기,장치의 따라서 화면을 다르게 표시할수 있습니다.

계단식 스타일 시트


캐스 케이 딩 스타일 시트 ( CSS은 )는 인 스타일 시트 언어 묘화에 사용되는 프리젠 테이션 에 작성하는 문서의 마크 업 언어 와 같은 HTML을 . [1]
CSS는 HTML 및 JavaScript 와 함께 World Wide Web 의 기본 기술입니다 . [2]
CSS는 레이아웃 , 색상 및 글꼴을 포함하여 프리젠 테이션과 컨텐츠를 분리 할 수 ​​있도록 설계되었습니다 . [3]
이러한 분리는 콘텐츠를 향상시킬 수있는 접근성 , 프리젠 테이션 특성의 사양에 더 많은 유연성과 제어를 제공, 여러 수 있도록 웹 페이지를 별도의 .css 파일에 관련 CSS를 지정하여 서식을 공유하고, 구조적 내용의 복잡성과 반복을 감소 . 서식과 내용을 분리하면 화면 상, 인쇄 중, 음성 (음성 기반 브라우저 또는 화면 판독기 사용 ) 및 점자 기반 (Braille-based ) 브라우저와 같은 다른 렌더링 방법에 대해 동일한 스타일의 동일한 마크 업 페이지를 표시 할 수 있습니다. 촉각 장치. CSS에는 모바일 장치 에서 콘텐츠에 액세스하는 경우 대체 서식 지정 규칙이 있습니다 . [4]
계단식 이름 은 지정된 우선 순위 스킴에서 유래하여 하나 이상의 규칙이 특정 요소와 일치하는 경우 적용 할 스타일 규칙을 결정합니다. 이 계단식 우선 순위 체계는 예측 가능합니다.
CSS 사양은 W3C ( World Wide Web Consortium )에서 관리합니다. RFC 2318 (1998 년 3 월)에 따라 CSS와 함께 사용하기 위해 인터넷 미디어 유형 ( MIME 유형 ) text/css이 등록됩니다 . W3C는 CSS 문서에 대해 무료 CSS 유효성 검사 서비스 를 운영합니다 . [5]
HTML 외에도 다른 마크 업 언어는 XHTML , 일반 XML , SVG 및 XUL을 비롯한 CSS 사용을 지원합니다 .
